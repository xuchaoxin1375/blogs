## vue 环境与安装

-   [Vue3+CLI](https://v3.cn.vuejs.org/guide/installation.html#%E5%91%BD%E4%BB%A4%E8%A1%8C%E5%B7%A5%E5%85%B7-cli)

    -   使用CLI安装:`yarn global add @vue/cli`

    -   创建一个模板项目:

        -   `vue create <yourNewVueProjectName>`

        -   安装项目依赖:

            -   `yarn install`

    -   运行vue项目:

        -   `yarn run serve`
